Тема: Разработать телеграмм бота


Описание: телеграмм бот для учета расходов. Пользователь создает свои категории

расходов. Команда /categories отрисовывает кнопками имена категорий, по нажатию на

кнопку с именем категории, нужно выбрать дату и пользователю отобразятся статьи

расходов. А так же появится кнопка чтобы добавить айтем в эту категорию этой датой.

Реализовать возможность удаления



Команда /total - возвращает сумму всех расходов пользователя

Команда /set_category - позволяет создать категорию расходов



Каждый день в 23:59 пользователь получает сообщение о расходах за сегодня в формате

YYYY-MM-DD

Category_1: n

Category_2 : k

Total: N + K

Технические требования:

Python 3.10 и выше

Пакет для тг бота https://github.com/python-telegram-bot/python-telegram-bot

Данные хранятся в базе данных

Приложение обернуто в docker compose

Инструкция по запуску в README.md

Код лежит в оформленом github репозитории

Токен бота берется из .env

Прикрепить диаграммы баз данных в случае использования реляционной БД
